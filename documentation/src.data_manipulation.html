Determines if there is a discontinuity between two points based on their positions and the time interval.

Parameters:

piDataFrame

The initial point of the entity, given as a section of a pandas DataFrame.

pfDataFrame

The final position of the entity, given as a section of a pandas DataFrame.

vmaxfloat or np.floating

The maximum allowable velocity for the entity.

Returns:

bool

True if there is a discontinuity, False otherwise.

Example Usage

    from src.data_manipulation.TRexDataCleaner import TRexDataCleaner

    dataCleaner = TRexDataCleaner()
    VMAX = 15

    if dataCleaner.isDiscontinuity(pi=unzippedData.loc[2], pf=unzippedData.loc[3], vmax=VMAX):
        unzippedData.drop(3)

In this example:

  • A constant VMAX is defined, specifying the maximum allowed threshold for detecting discontinuities.
  • The isDiscontinuity method of the TRexDataCleaner class is used to check if there is a discontinuity between two data points, pi and pf, at indices 2 and 3 in the unzippedData DataFrame. The method compares these points against the VMAX threshold.
  • If a discontinuity is detected, the data point at index 3 is dropped from the unzippedData DataFrame.
