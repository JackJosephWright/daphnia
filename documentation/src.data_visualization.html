Returns:

list

A list of all velocities calculated from the DataFrames

Example Usage

import numpy as np
import os
import pandas as pd
from src.data_visualization.avg_velocity import all_velocity, plot_histogram
                      
# directory containing csv table files 
direct_path = 'data/table_data'
                      
all_files = os.listdir(direct_path) 
dataframes = []
                      
# loop through csv files and read each one
for file in all_files: 
  file_path = os.path.join(direct_path, file) 
  df = pd.read_csv(file_path)
  dataframes.append(df)
# calculates velocities of all tables
all_velo = all_velocity(dataframes)
                      
# plots histogram of all calculated velocities
plot_histogram(all_velo)

In this example:

  • Necessary tools are imported, including numpy, os, and pandas, along with functions all_velocity and plot_histogram from the src.data_visualization.avg_velocity module.
  • The directory containing CSV files is specified, and all files in this directory are listed.
  • Each CSV file is read into a Pandas DataFrame and stored in a list called dataframes.
  • The all_velocity function is called to calculate velocities from all DataFrames.
  • The plot_histogram function is used to plot a histogram of the calculated velocities.

Splits valid data into tables

Parameters:

df: pd.dataframe

The input dataframe to be split

save_to_folder: bool, optional

If True, the resulting tables will be saved as CSV files in the specified folder (default is False)

folder_path: str, optional

The directory path where the CSV files will be saved, if save_to_folder is True

Returns:

List

A list of DataFrames, each representing a subset of the original DataFrame without infinite values

Example Usage

from src.data_visualization.split_table_dev import split_table
import pandas as pd
                      
clean_data = 'data/clean_fish_data/fish_data_clean.csv'
df = pd.read_csv(clean_data)
                      
# set save_to_folder to True if you want to create a folder of tables in explorer
tables = split_table(df, save_to_folder=False, folder_path='output_folder')
                      
# can choose index of what table you want to access
first_table = tables[0]
print(first_table)
                      
# prints number of tables
num_tables = len(tables)
print(num_tables)

In this example:

  • The `split_table` function from the `src.data_visualization.split_table_dev` module is used to split a Pandas DataFrame into multiple tables based on certain criteria.
  • The data is read from a CSV file located at data/clean_fish_data/fish_data_clean.csv into a Pandas DataFrame.
  • The `split_table` function is called with the DataFrame, and the option save_to_folder is set to False, meaning the tables are not saved as files but are returned as a list.
  • The first table in the list is accessed and printed, and the number of tables created is also printed.
